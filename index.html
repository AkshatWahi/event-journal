Help
Two Tabs
Plan
Set up before the event. Add companies & meetings to visit, write objectives, build your day plan, and save quick links. Tap a company or meeting to activate it — all Flow entries then go under that block.
Flow
Capture during the event. Type any text and press Enter for a quick note. Use / commands for structured entries. Each entry is grouped under the currently active company or meeting.

Quick Start
1Tap + New Event and name it — or try ☰ → Load demo event to explore all features with sample data
2In the Plan tab: add companies & meetings, set visit objectives, and build your day plan
3Switch to the Flow tab: tap a company or meeting to activate it, then start typing to log entries
4Use / commands (/photo, /contact, /todo…) for structured entries
5Tap ☰ → Export to save a PDF or full ZIP backup when done
6Use ☰ → Create Backup regularly to keep a safe copy
