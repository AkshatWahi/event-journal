Quick Start
1Tap + New Event and give it a name (e.g. "Embedded World 2026")
2Tap the event to open it
3Type any text and press Enter to log a quick note
4Use slash commands (/photo, /contact …) for structured entries
5Tap ⬆ Export to save a PDF or ZIP when done
6Use ☰ → Create Backup regularly to keep a safe copy
Install as App (PWA)
💡 A local HTML file cannot be installed as a PWA.
Android · Chrome
  1. Open the app URL in Chrome
  2. Tap ⋮ menu → Add to Home screen
  3. Choose Install (not "Create Shortcut") for full offline support
  4. Icon appears on home screen
iPhone · Safari
  1. Open the app URL in Safari (not Chrome)
  2. Tap the Share button ⎙
  3. Tap Add to Home Screen → Add
Desktop · Chrome or Edge
  1. Open the app URL
  2. Click the install icon ⊕ in the address bar
  3. Click Install — app opens in its own window
